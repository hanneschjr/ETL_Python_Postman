# ETL Python com o uso do Postman:
- Projeto tem como objetivo demonstrar o desenvolvimento de uma pipeline de dadados (ETL). Para tanto, o projeto se utilizará de uma API pública do Aeroporto de Schiphol, na Holanda, para obter os dados do tráfego aéreo daquele aeroporto, condensando todos em uma base de dados dessas informações em tempo real.
